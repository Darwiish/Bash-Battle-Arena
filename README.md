## Bash-Battle-Arena

Interactive Bash system menu with backup and configuration parser. Demonstrates progressive scripting skills through 15 levels, including file management, conditionals, loops, backups, and interactive menus.

## Features

- Arena setup with hero files and Battlefield scenarios

- Backup scripts with timestamped folders and rotation

- Configuration file parser (config.txt)

- Disk space monitoring

- Interactive system menu combining multiple tasks

- Logging and file organization (Archive, Victory_Archive, logs)

---

## Levels Overview

Covers progressive scripting challenges from basic file creation and loops to advanced menu-driven automation, backups, and system monitoring.

---

## Repository Structure

Includes directories like Arena, Arena_Boss, Backup, Battlefield, Victory_Archive, and logs, along with scripts for each level (level1_.sh to level15_.sh).
